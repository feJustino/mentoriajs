Hoisting

Hoisting uma maneira de pensar sobre como os contextos de execução funcionam em javaScript, que salva em memoria as variaveis na fase de compilação, assim podendo busca uma variavel sem antes declarar

example: bla = 2 var bla; // ... // é implicitamente entendido como: var bla; bla = 2; Var - O escopo depende onde a variavel foi declarada podendo ser global ou da função - Atribuir um valor a uma variavel não declarada a coloca num escopo global Let - são "hoisting" para começo do bloco em que elas são definidas Const
